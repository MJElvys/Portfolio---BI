# Portafolio de Business Intelligence – Elvys

Este repositorio reúne dashboards desarrollados en Power BI usando datos de distintos sectores:

- **Salud** – Indicadores hospitalarios y análisis de resultados clínicos.
- **Finanzas** – Segmentación y comportamiento de clientes bancarios.
- **Retail** – Ventas, stock y proyección de demanda.

El portafolio está en construcción.  
Los primeros entregables se estarán agregando progresivamente entre noviembre y enero.
